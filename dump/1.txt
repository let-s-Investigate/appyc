## GOAL
Add a complete, offline-capable PDF toolkit (iLovePDF-style) to ProScan. Every tool must run ON-DEVICE where technically possible (no upload/no internet required) and must be accessible from a single "Tools" hub grid with clear icons and labels.

## TOOLS TO IMPLEMENT (grouped)

### Scan & Capture (reuse existing camera)
- Scan to PDF: multi-page camera capture → single PDF, with auto-crop, filters (B/W, grayscale, color), and reorder before saving.
- Image to PDF: pick photos from gallery → combine into a PDF (adjust order, orientation, margins).

### Organize
- Merge PDF: combine multiple PDFs in a chosen order.
- Split PDF: extract a single page or a page range into new file(s).
- Organize Pages: reorder, add, and delete pages via a thumbnail grid (drag to sort).
- Rotate PDF: rotate selected or all pages; support batch.
- Page Numbers: insert page numbers with position, font, and size options.
- Crop PDF: crop margins or a selected area, apply to one page or the whole doc.

### Optimize
- Compress PDF: reduce file size with quality levels (Low/Recommended/High); show before/after size.
- Repair PDF: attempt to recover a corrupted/damaged PDF.
- OCR PDF: run on-device text recognition to make scanned PDFs searchable/selectable (respect device language packs).

### Edit
- Edit PDF: add text, images, shapes, and freehand annotations; edit font, size, color; move/resize elements.
- Watermark: stamp text or image watermark with position, opacity, rotation, and typography.
- PDF Forms: auto-detect form fields, fill them, and support creating basic fillable fields (text, checkbox, choice).

### Security
- Protect PDF: set a password / encrypt the document.
- Unlock PDF: remove a known password from a protected PDF.
- Sign PDF: draw/type/import a signature and place it; support self-sign flow (offline).
- Redact PDF: permanently remove selected text/graphics (true redaction, not just overlay).
- Compare PDF: side-by-side comparison highlighting differences between two versions.

### PDF Intelligence (AI — online allowed)
- AI Summarizer: generate a concise summary + key points from a PDF's text.
- Translate PDF: translate content while preserving layout, fonts, and formatting.

## EXCLUDED (do NOT build now)
- The entire Converter section (PDF↔Word/Excel/PowerPoint, HTML→PDF, PDF→JPG, PDF→PDF/A, PDF→Markdown).
